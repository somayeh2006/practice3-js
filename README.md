Random Product Suggestion
A simple JavaScript example that suggests a random product from a predefined list.
How it works
Stores product names in an array
Generates a random number using Math.random()
Rounds the number down with Math.floor()
Selects and logs a random product from the array
Purpose
Practice working with arrays
Understand random number generation in JavaScript
Learn how to select random items dynamically

(Practice 2)
Find Min and Max Values
A simple JavaScript example that finds the minimum and maximum values in an array of numbers.
How it works
Stores numbers in an array
Uses the spread operator (...) to pass array values to Math.max()
Finds the largest number in the array
Uses Math.min() to find the smallest number
Logs both results to the console
Purpose
Practice working with arrays
Learn how to use Math.min() and Math.max()
Understand the spread operator in JavaScript
